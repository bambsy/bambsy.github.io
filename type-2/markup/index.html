About:
This website was created for Typography II and to record the projects thus far. Using HTML and CSS we completed not only projects, but the coding for the webpage aswell. Please use the navigational links above to move through the webpage, as this is a work in progress you may notice changes in the appearance and setup of the website!

What most people don't realise is that coding isn't as easy and quick as we like to think. Before you can even start you have to set up a document so the browser can recognize that the file is a webpage. Then I set up some styles for the headers and fonts. Next is content, then I link everything together. Finally we hit 'save', and cross my fingers to hope I won't get any errors or else I have to go code hunting.
