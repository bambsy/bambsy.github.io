The main point of this project was to get us introduced to the different kinds of code and how they work. HTML was much easier to use and produces a much more flexible product. The web runs on HTML and leaves PostScript a bit behind, as its mostly only used with its intended programs. Both methods involve attention to opening and closing brackets, making sure none of the attributes overlap and make an error in the code, and both ask for time to iron out all the bumps.
